Program Control Design and Test Plan

Purpose
The program control component is responsible for coordinating the flow of the TopSecret program.
It interprets command-line arguments, requests information from the File Handler, applies deciphering when required,
and returns output to the user interface. This component does not directly access files or cipher keys.

Responsibilities
    Interpret command-line arguments passed to the program
    Determine which program mode to execute
    Request file lists and file contents from the File Handler
    Request deciphering from the Cipher component when applicable
    Handle errors and ensure the program exits gracefully

Program Execution Logic

Case 1: Program Executed With No Arguments
    Request a list of available files from the File Handler
    Assign a number to each file (01, 02, 03, etc.)
    Display the numbered file list to the user
    Exit the program

Case 2: Program Executed With One Argument
    Validate that the argument is a valid file number
    Request the corresponding file contents from the File Handler
    If ciphering is enabled, apply the default cipher key
    Display the file contents to the user
    Exit the program

Case 3: Program Executed With Two Arguments
    Validate the file number argument
    Accept the second argument as an alternate cipher key
    Request file contents from the File Handler
    Send the contents and key to the Cipher component
    Display the deciphered text
    Exit the program

Error Handling
    If an invalid file number is provided, display an error message
    If a requested file cannot be found, display an error message
    If deciphering fails, display an error message
    In all error cases, the program exits gracefully after displaying the message

Pseudocode

let args = command-line arguments

if number of args equals 0
request list of files
assign numbers starting at 01
print numbered list of files
exit program

if number of args equals 1
let fileNumber = args[0]

if fileNumber is not numeric
    print error message
    exit program

request file contents using fileNumber
decipher contents using default key
print readable contents
exit program

if number of args equals 2
let fileNumber = args[0]
let cipherKey = args[1]

if fileNumber is not numeric
    print error message
    exit program

request file contents using fileNumber
decipher contents using cipherKey
print readable contents
exit program

if number of args is greater than 2
print error message
exit program

TESTS

Verify program lists files when no arguments are provided
Verify program displays file contents when one argument is provided
Verify program displays file contents using alternate key when two arguments are provided
Verify program handles non-numeric file number
Verify program exits gracefully on invalid input
